Настройка доступа к БД выполняется в файле db_config.

В приложении Messenger планируется реализовать текстовый мессенджер на базе клиент-серверной архитектуры.
Клиент и сервер будут реализованы на базе модуля стандартной библиотеки Python socket.
Сервер будет создавать объект класса socket для прослушивания определенного порта и обработки запросов клиентов, приходящих на этот порт.
Клиент будет создавать объект класса socket для установления соединения (socket.connnection) с сервером и отправки запросов на:
1. регистрацию пользователя
2. добавления другого пользователя по его ID к своему списку контактов
3. удалению другого пользователя из своего списка контактов
4. отправки сообщений пользователю из списка контактов
5. запрос списка контактов
6. запрос истории сообщений
7. создание группового чата в роли администратора группы
8. добавление пользователей из своего списка контактов к групповому чату
9. отправка сообщений в групповой чат, участником которого является пользователь
